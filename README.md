# 🧞 지니 (Genie)

> **higplus 전용 AI 어시스턴트 프로젝트 허브**

---

## 📌 소개

이 레포지토리는 **지니(Genie)** — higplus 님의 개인 AI 어시스턴트가 관리하는 프로젝트 허브입니다.

모든 프로젝트, 워크플로우, 설정이 이곳에서 관리됩니다.

## 📂 구조

```
genie/
├── projects/          # 프로젝트 폴더
├── workflows/         # 자동화 워크플로우
├── configs/           # 설정 파일
├── docs/              # 문서 & 가이드
└── README.md
```

## 🚀 시작하기

```bash
# 클론
git clone https://github.com/highplus23-source/genie.git

# 이동
cd genie
```
